## Overview
The project focuses on detecting cyber-attacks in IoT/IIoT networks using machine learning techniques. The Edge-IIoT dataset was used to analyze network traffic and classify it as normal or malicious.

##  Steps Performed
- Data Loading from multiple CSV files
- Data Cleaning (handling duplicates and missing values)
- Feature Selection (removal of non-numeric columns)
- Dimensionality Reduction using PCA
- Model Training using:
  - K-Nearest Neighbors (KNN)
  - Random Forest
- Model Evaluation using Accuracy & Confusion Matrix
- Data Visualization

## Results
- KNN Accuracy: ~94%
- Random Forest Accuracy: ~92%
- PCA retained ~88% variance with 15 components

## Dataset
Edge-IIoTset Cyber Security Dataset (Kaggle)
